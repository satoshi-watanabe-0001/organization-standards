# 🚀 はじめに・初期セットアップ

プロジェクトを開始する際に必要な初期設定とガイドラインをまとめています。

## 📄 ガイド一覧

### DEVIN-INITIAL-SETUP-GUIDE.md
Devinの初期セットアップ手順書です。環境構築から基本設定までを網羅しています。

**対象者**: Devinを初めて使用する方、新規プロジェクトを開始する方

### DOCUMENT-USAGE-MANUAL.md
組織標準ドキュメントの利用方法を説明するマニュアルです。

**対象者**: 全メンバー

### ESCALATION-CRITERIA-GUIDE.md
問題発生時のエスカレーション基準を定義したガイドです。いつ、誰に、どのように報告すべきかを明確にします。

**対象者**: 全メンバー（特にプロジェクトリーダー、エンジニア）

---

## 🎯 推奨される読み方

1. **DEVIN-INITIAL-SETUP-GUIDE.md** で環境をセットアップ
2. **DOCUMENT-USAGE-MANUAL.md** でドキュメント体系を理解
3. **ESCALATION-CRITERIA-GUIDE.md** でエスカレーション基準を確認
