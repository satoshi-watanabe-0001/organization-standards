# ⚙️ CI/CDセットアップ

継続的インテグレーション・デプロイメントの設定ガイドとチェックリストをまとめています。

## 📄 ガイド一覧

### CI-SETUP-CHECKLIST.md
CI/CDセットアップの詳細チェックリストです。すべての設定項目を網羅しています。

**対象者**: CI/CDを初めて設定する方、詳細な設定を行う方

### CI-SETUP-QUICK-CHECKLIST.md
CI/CDセットアップのクイックチェックリストです。必須項目を素早く確認できます。

**対象者**: CI/CD設定経験者、素早い確認が必要な方

### CI-SETUP-LANGUAGE-MATRIX.md
プログラミング言語別のCI/CDセットアップマトリックスです。各言語特有の設定を参照できます。

**対象者**: 特定言語のCI/CD設定を行う方

### ci-setup-sql-quality-gate.md
SQLコードの品質ゲート設定に関するガイドです。

**対象者**: SQLを使用するプロジェクトのCI/CD担当者

---

## 🎯 推奨される使い方

### 初めてCI/CDを設定する方
1. **CI-SETUP-QUICK-CHECKLIST.md** で全体像を把握
2. **CI-SETUP-LANGUAGE-MATRIX.md** で言語特有の設定を確認
3. **CI-SETUP-CHECKLIST.md** で詳細設定を実施

### 既存のCI/CDを改善したい方
- **CI-SETUP-CHECKLIST.md** で漏れている項目をチェック

### SQLプロジェクトの方
- **ci-setup-sql-quality-gate.md** でSQL品質ゲートを設定
