# 🤖 AI活用ガイド

AI（Devin等）を効果的に活用するためのワークフロー、チェックリスト、プロンプト集をまとめています。

## 📄 主要ガイド

### AI-MASTER-WORKFLOW-GUIDE.md
AI活用の全体的なワークフローを解説するマスターガイドです。

**対象者**: AI活用を始める全メンバー

### AI-PRE-WORK-CHECKLIST.md
AI作業を開始する前に確認すべき項目をまとめたチェックリストです。

**対象者**: AI利用者全員

### AI-DELIVERABLE-REFERENCE-GUIDE.md
AI作業の成果物に関するリファレンスガイドです。

**対象者**: AI利用者、レビュアー

### AI-DOCUMENTATION-COMMENT-CHECKLIST.md
AIが生成するドキュメントとコメントの品質チェックリストです。

**対象者**: AI利用者、レビュアー

## 🎯 推奨される読み方

### 初心者向け
1. **AI-MASTER-WORKFLOW-GUIDE.md** で全体像を把握
2. **AI-QUICK-REFERENCE.md** でよく使う情報をチェック
3. **AI-PRE-WORK-CHECKLIST.md** で作業前確認

### 経験者向け
- **AI-PROMPT-TEMPLATES.md** で効率的なプロンプトを学習
- **AI-CODING-INSTRUCTIONS.md** でコーディング品質を向上
