# 要件分析書テンプレート

## 📋 このテンプレートについて

### 目的

このテンプレートは、JIRAのProduct Backlog Item (PBI)を起点として、実装前の要件分析を行う際に使用します。

### 使用タイミング

- **Phase 0: 要件分析**のStep 0.6で使用
- PBIの詳細な分析が必要な場合
- 実装開始前に要件を明確化したい場合

### 使用方法

1. このテンプレートをコピー
2. `[角括弧]`内の項目をプロジェクト固有の情報に置き換え
3. 不要なセクションは削除
4. 必要に応じてセクションを追加

### 関連ドキュメント

- `AI-MASTER-WORKFLOW-GUIDE.md` - Phase 0の詳細手順
- `00-guides/phase-guides/phase-0-requirements-planning-guide.md` - Phase 0詳細ガイド
